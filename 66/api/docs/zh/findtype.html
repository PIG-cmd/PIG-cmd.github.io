API请求时需要指定参数findtype 
这个参数可以取5个值number name donate dot star
这个参数通常和pid参数一起出现，关于pid的用法，请见下一章请求参数 - pid
如果这个参数没有的话，服务器会返回信息 API request error: The query format is not specified
如果这个参数的值错误的话，服务器会返回信息 API request error: The server could not find a matching database, check that the 'findtype' parameter in the request is correct
