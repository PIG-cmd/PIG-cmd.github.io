班费缴纳系统API是我们基于JavaScript开发的一套应用程序接口
因为Github的托管服务只支持静态页面，否则我可以用ASP写的
与其他的API不同的是，因为是静态页面，用户需要用GET方法访问指定的URL，并指定结果输出的哪个URL里
如果中途没有发送任何的错误
那么页面将会访问到指定的URL，并以描点的形式返回响应
