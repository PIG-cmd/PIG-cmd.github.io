API请求时除了需要参数findtype外，还需要一个参数pid
当findtype为number时，pid应该是学号
当findtype为name时，pid应该是名字
名字需要经过encodeURI()方法转化 
输入需要转化的字符串

当findtype为donate时，pid应该为数字（浮点数）
当findtype为dot或star时，pid应该为整数
如果pid不匹配的话，服务器会返回Database request error: The corresponding primary key could not be found in the database
注：上面英文注解为

findtype:查找类型
number:学号
name:姓名
donate:捐款
dot:点
star:星
